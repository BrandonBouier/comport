Change the conversation with transparency.

In 2015, Code for America partnered with Indianpolis, Indiana to explore using technology and city data to improve public safety. The result was this site, Comport, which makes it easier for law enforcement agencies to share data with the public. Initially, it covers citizen complaints, use of force, and officer-involved shootings.

In 2016, Code for America is expanding Comport to cover 3 more cities while broadening coverage to pursuits and assaults on officers.
