Dataset: Assaults on Officers

View this data on the Indianapolis Regional Data Portal

TK

Fields included in this dataset and their definitions:

Field Information
Id This is a hashed unique identifier for a given incident.
Officer Identifier This is a hashed identifier used to identify the officer within this data, for example to see if an officer has been involved in multiple assaults.
Service Type The reason for the interaction, such as Traffic Stop or Call for Service.
Force Type The type of weapon used against the officer.
Assignment The type of vehicle the officers were in.
Arrest Made True if the person assaulting the officer was arrested.
Officer Injured True if officer was injured during the incident.
Officer Killed True if officer was killed during the incident.
Report Filed True if a report was filed about the incident.

*Officer Call data represents the number of calls for service from residents that officers responded to plus the number of times officers themselves initiated a response.
